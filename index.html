Base94 Characters

Info

This password is built from 94 printable ASCII characters. It provides the most entropy per character, at ~7-bits/character. As such, it is the shortest ASCII password on this page.

Close

Base64 Characters

Info

Base64 strings encompass all lowercase and uppercase characters and all digits, including two additional characters. This follows RFC 4648. 6-bits/character.

Close

Base32 Characters

Info

Base32 strings can still be mostly unambiguous while using more characters from the keyboard. This follows RFC 4648. 5-bits/character.

Close

Base16 Characters

Info

Hexadecimal strings are unambiguous with digits 0-9 and characters a-f. However, they provide less entropy per character, so it needs to be longer.This follows RFC 4648. 4-bits/character.

Close

Base10 Characters

Info

Decimal strings are unambiguous with digits 0-9. However, they provide less entropy per character, at ~3-bits/digit, so it needs to be longer.
